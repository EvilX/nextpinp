# Auto PiP Manager

**Расширение для GNOME Shell** — автоматически управляет окнами режима «картинка в картинке».

[English version](README.md)

## Возможности

- Автоматически закрепляет PiP-окна **поверх всех остальных окон**
- Автоматически делает PiP-окна видимыми **на каждом рабочем столе**
- При появлении перемещает PiP-окно в **выбранный угол** экрана
- Нет иконки в верхней панели, нет фоновых процессов — только один слушатель событий создания окон

## Требования

- GNOME Shell 45 или новее

## Установка

```bash
git clone <repo-url>
cd NextPinP
./install.sh
gnome-extensions enable nextpinp@leonid.nasedkin
```

На **Wayland** после первой установки необходим перезаход в сессию.  
На **X11** можно перезапустить GNOME Shell без выхода: `Alt+F2` → `r` → `Enter`.

## Настройки

Открыть окно настроек через приложение **Расширения** или:

```bash
gnome-extensions prefs nextpinp@leonid.nasedkin
```

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Угол экрана | Угол, в который перемещается PiP-окно | Справа-Снизу |
| Отступ | Расстояние от угла экрана в пикселях (0–50) | 20 |

Доступные углы: Слева-Вверху, Справа-Вверху, Справа-Снизу, Слева-Снизу.

## Добавление нового языка

1. Скопировать `po/ru.po` в `po/<код>.po` (например, `po/de.po`)
2. Перевести строки `msgstr`
3. Запустить `./install.sh` — новый `.mo`-файл скомпилируется автоматически

## Структура проекта

```

```
